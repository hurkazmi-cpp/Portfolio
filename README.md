# Premium Developer Portfolio 🚀

A high-performance, minimalist portfolio designed for visual excellence and hardware-accelerated performance. Built with a "Vibe Coding" philosophy, leveraging modern web practices.

## ✨ Features
- **Dynamic Hero Section**: Hardware-accelerated mouse spotlight and drifting background orbs.
- **Vibe Terminal**: An interactive glassmorphic terminal gimmick that runs a real-time typing sequence upon load.
- **Project Showcase**: 4 featured projects with glassmorphic hover effects and smooth 3D tilt physics (Vanilla Tilt).
- **Fully Responsive**: Optimized for mobile, tablet, and desktop experiences.
- **Integrated Contact**: Functional email delivery via Formspree integration.
- **Bespoke UI**: Custom glassmorphic components and staggered entrance animations.

## 🛠️ Tech Stack
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (JIT Engine)
- **Animations**: CSS3 Keyframes & Hardware-accelerated Transforms (`translate3d`)
- **Icons**: Google Material Symbols & DevIcons

## 🚀 Deployment
This site is optimized for deployment on **GitHub Pages**, **Vercel**, or **Netlify**.
